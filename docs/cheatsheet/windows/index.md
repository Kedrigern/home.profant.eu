# Windows

## local user managment

Manage local users and groups with: `lusrmgr.msc`

## winget

`winget` is the Windows Package Manager, a command-line tool for managing software on Windows.

```console
winget install --source winget Mozilla.Firefox.cs Zed uv git vlc nvim python discord
winget upgrade        # list
winget upgrade --all  # upgrade all
```
