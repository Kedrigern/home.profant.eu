# Flatpak

https://flathub.org

```console
flatpak list                # list of installed flatpaks
flatpak install <repo> <app># install app from repo
flatpak update              # update all installed flatpaks
flatpak uninstall <app>     # uninstall app
flatpak info  <app>         # get info about app
flatpak repair <app>
```

## Installed apps

Installed apps via flatpak at Fedora 43:

- Headlamp
- Podman Desktop
- Gimp
- Dev Toolbox
